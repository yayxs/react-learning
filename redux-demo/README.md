## Action

`action`顾名思义`动作`，`行动` `行为`，一言以蔽之，它是把数据从应用传到`仓库`的一个动作，也就是这个数据仓库

- JS 对象

- 格式

  ```js
  {
      type:ADD, // type 字段是约定，大家的约定，表示动作的名字,
      index:1,
      content:'这是一点内容'
  }
  ```

  - 对象的`type`字段表示执行的动作;字符串常量
  - index 唯一的 ID,主要是一个标识
  - content（或者其他）JS 对象什么字段都可以啊，一个字段而已
